## 깃/ 깃허브/ 소스트리란?

- Git : 버전을 관리하기 위한 도구
- Source tree : Git을 편하게 쓸 수 있게 하는 도구
- GitHub 
    - 원격 저장소 호스팅 서비스
    - 원격 Git으로 관리한 프로젝트 호스팅 서비스
    - 인터넷 상에서 깃으로 관리한 프로젝트를 관리해주는 서비스
    - 개발자들의 SNS
