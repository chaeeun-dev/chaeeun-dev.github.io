- template
    ###함수 탬플릿과 클래스 템플릿
    ```cpp
    template<typename T>
    void A(T a)
    {

    }

    // 여러 개 사용해도 됨
    template<typename T1, typename T2>
    void A(T1 a, T2 b)
    {
	    std::cout << a << " " << b << std::endl;
    }

    // 템플릿 특수화 - 특정 자료형인 함수를 호출하고 싶을 때
    template <>     
    void A<int a>
    {

    }
    ```
 
    템플릿은 여러가지 버전을 통용하는 함수가 만들어지는 게 아니다. 각기 다른 별도의 함수가 만들어 지는 것이다!
