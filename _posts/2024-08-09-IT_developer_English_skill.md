## 24.08.09

## Ch0. IT 영어, 어떻게 공략할까?
    
어딘가에서 이 악순환의 고리를 끊고 영어와 정면으로 부딪혀야 한다.

아직 자동 번역 결과의 품질은 충분히 높지 않다.

해외건 국내에서 일하건 업무에 필수적인 것은 ‘영어 읽기 능력’

우선순위를 정해서, 국내에서 일하는 개발자라면 먼저 영어 읽기부터 시작할 것을 추천, 물론 여유가 있다면 영어 말하기나 듣기 실력도 높이는 노력을 해야 됨.

기업에서는 젊은 인재에게 영문 읽기 능력을 원함.

꾸준히 단어를 외우고 어휘력을 높이자.

이 책을 통해 기르는 능력 - 어휘력과 표현, 도구, 테크닉, 담화 구조 파악

읽기, 쓰기, 듣기를 모두 함께 습득할 수 있도록 목표를 세워라.
    
## Ch1. 영어 읽기에 필요한 지식과 테크닉

1. 읽기에 필요한 4가지 기둥
- 어휘 - 영어 단어를 얼마나 알고 있는가
- 문법 - 문장을 구성하는 규칙에 관한 지식
- 담화 구조 - 각 문서 유형 고유의 구성 패턴
- 전문 지식 - 문서 내용에 관해 알고 있는가

cf) e.g. == 예를 들어, i.e == 즉 ← 라틴어

2.영어 읽기의 기초는 어휘다
- 암기 - 자신이 주로 읽는 문서에 자주 나오는 단어를 기억해 두는 것이 효과적. IT 분야의 영단어는 한정적임!
- 사전 활용
3.IT영어 읽기를 위해 알아야 할 영문법
- 단어 < 구 < 절 < 문장
- 문맥상 아는 어휘는 생략됨 - 문맥을 통해 알 수 있기에 주어, 동사, 관사, 목적어 등을 생략함 e.g. Delete from my phone, Network Unable → UI, 커밋 메시지에서 흔함!
- 설명이나 순서를 설명할 때 사용되는 콜론(:)
    
4.문장 전체의 담화 구조에 주목하자
  
담화 구조 : 하나 이상의 문장이 유기적으로 연결되어 일관된 내용을 형성하는 구조 (문법X)

담화 표지 - however, in other words 등으로 담화 구조의 흐름 파악

- 순서 열거 - first, second, thrid, then, finally
- 강조 - 사실, 실제로 : in fact, actually / 특히, 그중에서 : specifically, especially, particularly / 아무튼 : anyway
- 추가 - 또한 : also / 덧붙여, 게다가 : inaddition, besides / ~뿐만 아니라 …도 : not only ~ but also … / 반면 : in the other hand
- 역접 - but, however / ~지만 … : although~, …, though ~, …  / ~라고 하더라도 … : even if ~, …
- 예시- for example, for instance, such as, e.g / ~도 포함해서 : including
- 다른 말로 바꾸기 - 다시 말해 : in order words / 즉 : namely, ie, or
- 이유 - because ~, since ~, as ~ / because of ~, due to ~
- 결론, 결과 - 그래서, 따라서 : so, therefore, thus / 그 결과, 따라서 : as a result, consequently / 요약하면 : in summary, in short
